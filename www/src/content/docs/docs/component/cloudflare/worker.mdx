<Section type="about">
The `Worker` component is a higher level component that makes it easy to create a Cloudflare Worker.

#### Using the minimal config
```ts
new sst.cloudflare.Worker("MyWorker", {
  handler: "src/worker.handler",
});
```
</Section>
