<Section type="about">
The `Postgres` component is a higher level component that makes it easy to create a Vector database powered by RDS Postgres Serverless v2.

#### Using the minimal config
```js
new sst.aws.Postgres("MyDatabase");
```
</Section>

## ScalingProps

Aurora Serverless v2 scaling configuration
<InlineSection>
- <p><code class="key">max?</code><code class="primitive">Input</code><code class="symbol">&lt;</code><code class="primitive">number</code><code class="symbol">&gt;</code></p>

<InlineSection>
**Default** - 4
</InlineSection>
The maximum number of Aurora capacity units (ACUs) for a DB instance in an Aurora Serverless v2 cluster.
