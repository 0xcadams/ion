---
title: Use Next.js with SST
description: Create and deploy a Next.js app to AWS with SST and OpenNext.
---

We are going to create a Next.js app, add an S3 Bucket for file uploads, and deploy it to AWS using SST.

Before you get started, you want to:

1. [Configure your AWS credentials](https://docs.sst.dev/advanced/iam-credentials#loading-from-a-file)

2. ```bash title="Install SST"
   curl -fsSL https://ion.sst.dev/install | bash
   ```

:::note
The CLI currently supports macOS, Linux, and WSL.
:::

---

## 1. Create a Next.js app

Let's start by creating our app.

```bash
npx create-next-app@latest
```

1. Init SST

1. `sst dev next dev`

1. Add bucket to `sst.config.ts`

1. Bind to Next.js

1. Generate pre-signed URL in Next.js

1. Render upload form

1. Test upload

1. `sst deploy`

1. Test deployed site

1. Open console and sign up

1. Connect AWS account

1. View logs from Next.js
