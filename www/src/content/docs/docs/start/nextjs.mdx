---
title: Use Next.js with SST
description: Create and deploy a Next.js app to AWS with SST and OpenNext.
---

Prerequisites

- Configure AWS credentials

Steps

1. Create Next.js app

1. Install SST

1. Init SST

1. `sst dev next dev`

1. Add bucket to `sst.config.ts`

1. Bind to Next.js

1. Generate pre-signed URL in Next.js

1. Render upload form

1. Test upload

1. `sst deploy`

1. Test deployed site

1. Open console and sign up

1. Connect AWS account

1. View logs from Next.js
